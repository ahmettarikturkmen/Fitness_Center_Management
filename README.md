PROJE ADI: Fitness Center Yönetim Sistemi
DERS     : Web Programlama (2025-2026 Güz Dönemi)

1. PROJE HAKKINDA
---------------------------------------------------------
Bu proje, bir spor salonunun günlük operasyonlarını dijitalleştirmek amacıyla 
geliştirilmiş bir Web Yönetim Sistemidir. Sistem; üyelik işlemleri, antrenör 
takibi, hizmet yönetimi ve randevu alma süreçlerini kapsamaktadır.

2. KULLANILAN TEKNOLOJİLER
---------------------------------------------------------
- Backend: ASP.NET Core MVC (C#)
- Veritabanı: PostgreSQL
- ORM: Entity Framework Core (Code First)
- Kimlik Doğrulama: ASP.NET Core Identity
- Frontend: HTML5, CSS3, Bootstrap 5, JavaScript

3. TEMEL ÖZELLİKLER
---------------------------------------------------------
- Üyelik Sistemi: Kullanıcı kayıt (Register) ve giriş (Login) işlemleri.
- Rol Yönetimi: Admin ve Üye (Member) yetkilendirmeleri.
- Randevu Modülü: Üyelerin uygun eğitmen ve hizmet için randevu alabilmesi.
- Eğitmen ve Hizmet Yönetimi: Salon hizmetlerinin ve eğitmenlerin listelenmesi.
- Veri Doğrulama: Server ve Client tabanlı validasyonlar.
